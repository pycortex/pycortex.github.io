cortex.database.Database.get_view¶

Database.get_view(self, vw, subject, name)[source]¶

Set the view for an open webshow instance from a saved view

Sets the view in a currently-open cortex.webshow instance (with handle vw) to the saved view named name

Parameters
vwhandle for cortex.webshow

Handle for open webshow session (returned by cortex.webshow)

subjectstring, subject name
namestring

Name of stored view to re-load

Notes

Equivalent to call to vw.get_view(subject,name) For a list of the view parameters saved, see viewer._capture_view
